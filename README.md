# 🎓 Career Guidance Web Application

A modern and responsive career guidance platform built to help students explore the best career paths after 10th grade. This application provides detailed insights into academic and skill-based streams, helping students make informed decisions about their future.

---
## 🚀 Features

- 🔍 Explore multiple career paths (Science, Commerce, Arts, IT, etc.)
- 🗺️ Detailed career roadmaps for each stream
- 📚 Course information with duration, fees, and salary insights
- 🌟 Success stories of inspiring personalities
- 🎯 Clean and user-friendly UI
- 📱 Fully responsive design

---
## 🛠️ Tech Stack

- **Frontend:** React.js
- **Build Tool:** Vite
- **Styling:** CSS
- **Routing:** React Router

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/career-guidance-app.git

# Navigate to project folder
cd career-guidance-app

# Install dependencies
npm install

# Run the development server
npm run dev
